## Requirements

- Latte dock git master branch
- Latest or updated Qt and Plasma frameworks and libraries.

## Installation

1. Just `cd` to the cloned repo.
2. Execute ```$ kpackagetool5 -i . -t Latte/Indicator```.
3. Set the indicator theme to `Glorious`.
